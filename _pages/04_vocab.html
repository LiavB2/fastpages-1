--- layout: page title: Vocab permalink: /vocab/ --- Throughout the year, I will put new vocabulary terms that I learn in APCSP here!
  • Procedural abstraction: grouping a sequence of commands, often used repeatedly
  • Application Programming Interface (API): Exchange of data between programs includes the data type the method/function and the format of the data type. These concepts are key to learning how to write functions, receive, and return data.
  • Recursion: something that clals itself repeatedly.
